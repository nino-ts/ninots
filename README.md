# Ninots Application

A Next.js + Laravel + Bun hybrid application.

## Getting Started

```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Start production server
bun run start

# Build for production
bun run build

# Run CLI commands
bun run nino --help
```
